CREATING TESTS:

Each test file MUST have:

1. Test function with clear name
2. Expected output documented
3. Actual output captured
4. Pass/fail comparison
5. Error handling

Example test structure:
```python
def test_memory_write_read():
    """
    Test: Memory write followed by read
    Expected: Read returns same data that was written
    """
    # Setup
    from tools.tools_memory import MemoryTools
    memory = MemoryTools(SUPABASE_URL, SUPABASE_KEY)
    
    # Test
    test_data = "Test content for ticket 001"
    
    # Write
    write_result = memory.memory_write(
        content=test_data,
        tier="L2",
        importance=3,
        category="test"
    )
    
    # Verify write succeeded
    assert write_result.get("status") == "success", f"Write failed: {write_result}"
    
    # Read
    read_result = memory.memory_read(query="Test content")
    
    # Verify read succeeded
    assert len(read_result) > 0, "Read returned empty"
    assert test_data in str(read_result), f"Test data not found in read results: {read_result}"
    
    print("✓ Test PASSED: memory_write_read")
    return True
```
